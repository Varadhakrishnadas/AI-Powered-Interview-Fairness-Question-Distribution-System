# Interview Fairness & Question Distribution System

## Problem

In interviews, candidates often get questions of different difficulty levels, which is unfair.

## Solution

This system distributes questions equally among candidates and checks fairness automatically.

## Features

* Upload questions (CSV/PDF)
* Automatic difficulty detection
* Fair distribution of questions
* Fairness alerts
* PDF download

## Tech Used

* Python
* Streamlit
* Scikit-learn
* SQLite

## How to Run

pip install -r requirements.txt
streamlit run main.py
